# IoT Water Meter Monitoring Device

A device can monitor water meter real-time with hall and temperature sensor.

## Descripton

`main-on-resberry.py` only works on **Resberry Pi**.

`Sensor/` includes PCB board Gerber files, and the sensor includes **hall sensor** with **HAL3144UA** and **temperature sensor** with **TMP102**.

`static/` `templates/` are dirs built by `test_flask.py`.
